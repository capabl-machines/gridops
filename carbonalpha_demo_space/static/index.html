How CarbonAlpha works

A 7B-parameter language model that reads one macro headline today, then commits a single 3-year portfolio across five real ETFs — designed to outperform an equal-weight benchmark on both portfolio returns and carbon emissions.

01

The challenge

Build a portfolio agent that optimises returns while favouring carbon savings, navigating global macro events through coherent macro reasoning.

Goal: outperform the equal-weight benchmark on both real return and cumulative carbon across a 12-quarter macro path.

02

Long-horizon thinking

CarbonAlpha can plan for full 3 year cycle and navigates itself in any unforeseen scenario.

So it has to reason at the macro-cycle level: identify the 1st-order direct hit, the 2nd-order chain (capital flows, sector rotation), and the 3rd-order counterintuitive move that pattern-matchers miss.

03

Re-planning Capability

If new global news arrives, the agent can re-plan upcoming quarters. Every re-plan carries turnover costs and carbon consequences

04

How it was trained

Qwen 2.5 7B base → SFT on a 400-trace curriculum (easy / ambiguous / hard shocks) → GRPO reinforcement-learning refinement (Phase 1, 100 steps).

SFT teaches the format and the macro-cycle frame. GRPO refines toward the actual reward: high real return × low carbon × low drawdown × clean output format.

05

What you can do here

Click Plan Portfolio. The model reasons live, word by word. The same prompt runs through three versions side-by-side: A · GRPO (final), B · SFT (warm-start), C · base Qwen (untouched). You see the training delta directly.

Edit any quarter's headline and re-plan — every quarter from there forward re-projects from the new allocation. The carbon thermometer, NAV vs benchmark, and Reward Breakdown update live.
